# Calculadora Científica Temática

Uma calculadora científica desenvolvida com HTML, CSS e JavaScript, com suporte a múltiplos temas e diversas funções matemáticas.

## 🚀 Funcionalidades

- Operações básicas (+, -, ×, ÷)
- Funções científicas:
  - sin, cos, tan
  - log, ln
  - raiz quadrada e cúbica
  - potência (x², x³, xʸ)
  - fatorial (!)
- Constantes matemáticas (π, e)
- Alternância entre modo normal e científico
- Sistema de temas (dark, azul, rosa, etc)
- Interface responsiva

## 🎨 Tecnologias

- HTML5
- CSS3 (variáveis + temas)
- JavaScript (lógica da calculadora)

## 💡 Objetivo

Projeto criado para praticar lógica de programação, manipulação do DOM e construção de interfaces interativas.
